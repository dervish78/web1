list tag

list 태그는 선택 리스트를 만들때 사용합니다.
list 테그는 (ol) 태그과 (ul) 태그와 같이 사용됩니다.



Result

1. An ordered list 는 번호를 자동으로 카운트하는 방법이며, 중간 번호가 삭제되는 경우 번호를 재 정렬합니다.
2. add number ordered list 는 1번 과정에 번호를 직접 넣어본 것으로 번호가 2개가 표시되는 것을 볼 수 있습니다.
3. An unordered list 는 목록만 구분합니다.
4. add number An order list 는 3번 과정에 번호를 직접 넣어본 것으로 번호와 점이 같이 표시되는 것을 볼 수 있습니다.
